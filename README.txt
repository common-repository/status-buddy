== Description ==

Install plugin to availability of all buddypress members 
This plugin require <a href="https://wordpress.org/plugins/buddypress/">Buddypress</a> already installed.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.
1. Upload `status-buddy.zip` to the `/wp-content/plugins/` directory and extract it.
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Is it show live status? =

it will update status in 5 seconds of delay. 

= What is the formula for Away status? =

If there is no activity for user till 5 minutes, it will be considered as an away. 


= What is the formula for offline status? =
If there is no activity for user till 30 minutes or logged out, it will be considered as an away.
